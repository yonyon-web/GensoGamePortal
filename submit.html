🎮 ゲームをポータルに登録する

← ポータルに戻る

元素騎士ミニゲーム開発キットで作ったゲームをポータルに掲載できます。
登録は GitHub の Pull Request で行います。

前提条件

ゲームが GitHub Pages で公開済みであること。
まだの場合は開発キットの docs/deploy-guide.md を参照してください。

登録手順

1
このリポジトリを Fork する

GitHub で「Fork」ボタンを押して自分のアカウントにコピーします。

2
games.json にゲーム情報を追記する

配列の末尾に以下の形式で追加してください。

{
  "id": "YYYYMMDD-yourname",
  "title": "ゲームのタイトル",
  "creator": "GitHubユーザー名",
  "creatorUrl": "https://github.com/あなたのユーザー名",
  "url": "https://あなたのユーザー名.github.io/リポジトリ名/",
  "thumbnail": null,
  "collections": ["equipment_v2"],
  "tags": ["battle"],
  "description": "ゲームの説明を1〜2行で",
  "submittedAt": "2025-04-20"
}

tags の選択肢: battle / puzzle / collection / casual

collections の選択肢: equipment_v1 / equipment_v2 / block / item

3
Pull Request を作成する

Fork したリポジトリから元のリポジトリへ PR を出します。タイトルは Add game: ゲームタイトル の形式にしてください。

4
レビュー後、自動デプロイされる

管理者がマージすると GitHub Actions が自動的にポータルを更新します。

💡 開発キットを使っている場合
Claude Code で /submit-portal と入力すると、AIが自動でこれらの手順を実行してPRを作成します。
